# Contributing to MOBIUS

Dziękujemy za zainteresowanie projektem.

## Jak przyczynić się do rozwoju

1. **Fork** repozytorium
2. **Branch** — utwórz gałąź: `git checkout -b feature/nazwa-funkcji`
3. **Commit** — zmiany: `git commit -m "Dodaj: opis"`
4. **Push** — wypchnij: `git push origin feature/nazwa-funkcji`
5. **Pull Request** — otwórz PR na GitHub

## Zasady

- Kod w Pythonie 3.11+
- Zachowaj styl: lapidarny, bez zbędnych komentarzy
- Testy mile widziane (pytest)
- Dokumentacja: aktualizuj README przy nowych funkcjach

Zobacz [MOBIUS_ROADMAP.md](MOBIUS_ROADMAP.md) — lista planowanych funkcji.
